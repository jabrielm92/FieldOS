# FieldOS

FieldOS is a multi-tenant Revenue & Operations platform for field-service companies.

## Quick links
- Canonical runbook: `docs/RUNBOOK.md`
- Backend env template: `backend/.env.example`
- Frontend env template: `frontend/.env.local.example`
- Product/history context: `requirements.md`

## Fast start
```bash
docker compose up --build
```

Then open `http://localhost`.
